# Sliver NFT Game (Website)

**Link:** [sliver.world](http://sliver.world)
**Technology Used:** React, TailwindCSS, Nodejs, Context API

---

## Overview
Sliver is a multi-platform 1v1 battle game. Build your team, choose your playstyle, and compete for rewards. Each character has unique skills and stats. Equip weapons, use element potions, and rank up to become the King of Sliver.
