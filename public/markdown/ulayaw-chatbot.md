# Ulayaw Web-Based Chatbot

**Link:** [ulayaw.herokuapp.com](http://ulayaw.herokuapp.com)
**Technology Used:** React, MongoDB, TailwindCSS, Heroku, OpenAI-GPT

---

## Overview
Ulayaw is a web-based chatbot that assesses suicidal behaviors using the SBQ-R. Using openai-gpt it provides responses based on schema-guided dialogue and uses location-based services to ensure critical cases get immediate attention.
